4) How to map & prioritize these keywords (quick roadmap)
    1. Homepage: Primary short-tail + branded phrases (e.g., “Mission Mind Organization — humanitarian organization Afghanistan”, “Donate to education & WASH programs in Afghanistan”).
    2. Programs pages: One main mid-tail per program (Education page → “community based education Afghanistan”), plus 4–6 long-tails as H2s/FAQ entries.
    3. Resources/Reports: Use transactional/resource keywords (e.g., “MMO annual report PDF”, “RFP NGO Afghanistan”).
    4. Donate page: Transactional long-tails (“how to donate to an NGO in Afghanistan”, “donate now MMO”).
    5. Jobs page: “NGO jobs in Kabul”, “apply MMO job Afghanistan 2025”.
    6. Blog/News: Use long-tail educational content and case studies (“WASH project case study Afghanistan”, “how NGOs support GBV survivors in Afghanistan”).
    7. Local landing pages: Create localized pages for each province and use geo keywords.
5) Implementation tips (quick)
    • Put main keyword in page title, H1, URL slug, meta description and first paragraph.
    • Use long-tail phrases naturally in subheadings and FAQ sections.
    • Add structured data for organization, events, jobs and publications.
    • Translate top pages into Dari and Pashto (hreflang) for local reach.
    • Add ALT text to images using keywords (e.g., “WASH project MMO Balkh 2025”).
    • Create resource PDFs (annual report) with crawlable text and link them with keyworded anchor text (“Download MMO annual report 2025 (PDF)”).
